💬 Chat App
A modern chat application built with React and Tailwind CSS.

Features
💬 Real-time messaging

🌙 Dark/Light mode

👤 User avatars

⌨️ Typing indicators

📁 File sharing

😊 Emoji picker

📱 Responsive design

Quick Start
bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
